# microq
A tiny Websocket-based event broker.

## Features
- [x] Grouped subscriptions
- [ ] Offline messages
- [ ] Clustering
- [x] QoS (0, 1)
- [x] Topc Patterns

### Topic patterns
Topic patterns are heavily inspired by [NATS's subject-based messaging](https://docs.nats.io/nats-concepts/subjects).
